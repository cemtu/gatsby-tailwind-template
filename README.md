---

## Installation

To run project on your local:

```bash
$ git clone https://github.com/BayCem/gatsby-tailwind-template
$ yarn install

```

Then, run the following command for run project in local server:

```bash
$ gatsby develop
```

or for run in production:

```bash
$ gatsby serve
```
